# Day 12: Hot Springs

## Problem

给你一行包含 `.`, `#`, `?` 的字符串, 其中 `?` 表示待定可能为 `.` 中的 `#` 一个. 把这一行中的连续的 `#` 长度记成一个数组, 数一下满足多少种问号的分配方式满足指定 `#` 长度数组.

`#` 长度数组的例子:

```text
#.#.### 1,1,3
.#...#....###. 1,1,3
.#.###.#.###### 1,3,1,6
####.#...#... 4,1,1
#....######..#####. 1,6,5
.###.##....# 3,2,1
```

## Solution

DP 一下, 记 `f[n][len][matched]` 表示, 当前匹配了字符串的前 n 个字符, 当前连续段的长度为 len, 已经匹配好了 matched 个段, 转移显然.
